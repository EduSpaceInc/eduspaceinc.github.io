Welcome to GitHub Project home. EduSrch is a formula search engine where students can search, print, calculate and get explanation of a formula they wanted to. For that reason on calculate page we are using JavaScript so as to get instant calculation results. At here you can contribute to EduSrch by coding a little bit for a specific formula. You are the valuable contributor to us and last but not the least EduSpace Inc Welcomes You...!

Before you begin check for remaining pages which do not have calculate option. Select only that Number/formula which has RED bubble infront of it. Where number represents formula id and in order to get which formula it is you can click on that number. After you create HTML page you can check here JavaScript Codes for each formula. Also you can review other contributor's work. If any discripancy found you can modify at repo. In order to Varify your submission go to "eduspaceinc.github.io/edusrch/41.html. Here 41 is the Formula id and extension you save is .html."
